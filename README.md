# Open Source Internet Safety Pamphlet

This is a standardized document made for people in Toledo, Ohio but can be used anywhere.

## How to use?

Download the PDF in this repository/folder and print away!

## Contributing

Pull requests are welcome. For major changes, please open an issue first
to discuss what you would like to change. To edit any part of this document, simply open the ODG file in Libre Office.

Please make sure to update tests as appropriate and export to PDF for re-commiting.
